Welcome

DirHash is a console program that computes the hash of a given directory content. Lexicographical order is used for browsing the directory hierarchy. Supported hashing algorithms are SHA-1, SHA-256, SHA-384 and SHA-512.

Usage

DirHash can be invoked from the command like as follows:

DirHash.exe DirectoryOrFilePath [HashAlgo] [-t ResultFileName] [-nowait]

Possible values for HashAlgo (not case sensitive):

  • SHA1
  • SHA256
  • SHA384
  • SHA512

If HashAlgo is not specified, SHA-1 is used by default.

ResultFileName specifies an optional text file where the result will be appended.

if -nowait is specified, program will exit immediately after displaying the hash result. Otherwise, it prompts user to hit a key before it exits.

BUILDING FROM SOURCE

DirHash source code comes with a Microsoft Visual C++ 2010 project file, but you can use the free Microsoft Visual Studio Community 2013 to build it as well.

DirHash uses OpenSSL for its cryptographic operations and the source comes with pre-built library files of OpenSSL 1.0.2c
